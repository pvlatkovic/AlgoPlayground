
stack api:

pop() - remove top item from the stack
push(T) - add an item to the top of the stack
peek() - return the top of the stack
IsEmpty() - return true if and only if the stack is IsEmpty

---------------------

queue api:

add(T) - add an item to the end of the list
remove() - remove the first item in the list
peek() - return the top of the queue
IsEmpty() - return true if and only if the queue is IsEmpty
